# Custom Smart Pointers Library (C++11/14/17)

[![CI](https://github.com/VSRom/smart_ptrs/actions/workflows/ci.yml/badge.svg)](https://github.com/VSRom/smart_ptrs/actions)

## Реализовано

*  Использование класса **`atomic`** для использования в многопоточности.

*   **`myUnique_ptr`** — указатель с монопольным владением ресурсом. Копирование запрещено, поддерживается только эффективное перемещение (`Move`).
*   **`myShared_ptr`** — указатель с совместным владением ресурсом. Память автоматически очищается, когда уничтожается последний владелец. Безопасен к утечкам благодаря идиоме **Copy-and-Swap**.
*   **`myWeak_ptr`** — слабый указатель-наблюдатель, предотвращающий циклические зависимости. Имеет метод `lock()` для временного получения сильной ссылки и контроля времени жизни данных.
